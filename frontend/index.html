Personal Storage Cloud

Welcome to your self-hosted ecosystem running on Debian Linux.

OS: eMMC (15GB)
Storage: USB sda1 (14GB)
RAM: 4GB

What is this?

This is a private, high-performance file server built with Go (Golang) and Gin. By offloading data to external USB storage, we keep the system lightweight and modular.

Current Architecture:

  • Backend: Go Binary (Compiled for speed)
  • Database: SQLite (Stored on USB via Symlink)
  • Frontend: HTML5/JS (Embedded directly in the app)
